<h2 align="left">GradeTracker.</h2>

###

<p align="left">A full stack web application for tracking student grades across multiple subjects, built as a hands-on learning project to understand how modern web applications are structured and how their components communicate.</p>

###

<h4 align="left">About the project</h4>

###

<p align="left">GradeTracker allows users to add students, assign them to subjects, and record grades against each subject. The application stores all data in a real PostgreSQL database and exposes it through a REST API that a React frontend consumes.<br>The project was intentionally kept simple in scope so that the focus could remain on understanding the architecture and the relationship between each layer of the stack, rather than on building complex features.</p>

###

<h4 align="left">Tech stack</h4>

###

<p align="left">Frontend: React (Vite)<br>Backend: ASP.NET Core Web API (C#)<br>Database: PostgreSQL<br>ORM: Entity Framework Core<br>API Documentation: Swagger UI</p>
